# Game Editor Shortcuts

## @shortcuts

### Window Controls
- Ctrl+N: New Project
- Ctrl+O: Open Project
- Ctrl+S: Save Project
- Ctrl+Shift+S: Save As

### Edit
- Ctrl+Z: Undo
- Ctrl+Y: Redo
- Ctrl+X: Cut
- Ctrl+C: Copy
- Ctrl+V: Paste
- Ctrl+H: Find & Replace

### Navigation
- Ctrl+G: Go to Line
- F5: Refresh View
